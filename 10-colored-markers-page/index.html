Color Formats Demo

This HTML and CSS project demonstrates different ways to define and apply colors using named colors, HEX values, RGB, RGBA, HSL, and HSLA. It renders a set of colored markers (red, green, blue) with caps, sleeves, and gradients. We use linear-gradient() for smooth transitions and box-shadow with alpha for depth. Transparency is shown via RGBA and HSLA to control opacity independently of hue, saturation, and lightness.
